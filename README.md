# Account Deletion Page — Digital Document Wallet

## Deploy to GitHub Pages

```bash
cd E:\Personal\account-deletion-site
git init
git add index.html
git commit -m "Add account deletion page"
git remote add origin https://github.com/YOUR_USERNAME/ddw-account-deletion.git
git push -u origin main
```

Then: repo **Settings → Pages** → branch `main`, root `/` → Save.

URL: `https://YOUR_USERNAME.github.io/ddw-account-deletion/`

## Deploy to Render

1. Go to [render.com](https://render.com) → New **Static Site**
2. Connect your GitHub repo
3. Publish directory: `.`
4. Deploy

URL: `https://ddw-account-deletion.onrender.com`

## Google Play Console

Paste the deployed URL into the **Delete account URL** field.
